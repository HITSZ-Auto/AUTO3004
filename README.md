## 关于考试

- 课程知识密度并不大，很多计算量很大的内容在平时的编程中实现过了，所以不会考。而且由于你校众所周知的阉割传统，很多容易的内容被不断重复，难的内容都会被砍掉。
- 考试难度：背诵+简单计算，龙格库塔法可能步骤多一些
